Twitter Bootstrap

A lot of "Grunt" work is already done for you and lets get started on what you really want to do. All of the CSS for layouts, handling responsive elements, modal windows, etc is there for you to use with little effort. It makes for a great starting point for something like a data entry tool.

jQuery

Twitter Bootstrap already makes use jQuery so it makes sense to leverage it for other features. I've used it here to help me populate dropdowns dynamicly based on user selections, creating events to help manage the buttons and other actions of the page.

jQuery - UI (DatePicker)

jQuery - UI provides many useful components and the one I am making use of is the Date Picker. This retricts user entry to values that are suitable to me and provides a simple familiar interface for the user.

jQuery - Plugin - TimePicker

This TimePicker plugin for jQuery provides very similar functionality plus a similar look & feel to the DatePicker. It also retricts the values that can be entered into the field so I will always have a consistent and usable value.
